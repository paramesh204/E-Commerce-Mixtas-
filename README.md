🛒 Mixtas E-Commerce (React)

A modern and responsive E-commerce web application built using React.js, featuring dynamic product handling, cart functionality, and interactive UI with event-driven behavior.

🚀 Features

🛍️ Product listing with responsive design

❤️ Add to cart using event handlers

🔍 Search functionality for products

🧮 Dynamic cart (quantity & subtotal update)

📱 Fully responsive UI (mobile + desktop)

⚡ Fast and smooth user experience

🧠 Event-driven interactions (click, input, key events)


🛠️ Technologies Used

⚛️ React.js (Hooks: useState, useEffect)

🎨 CSS / Bootstrap / Tailwind (based on your setup)

🧩 JavaScript (ES6+)

🌐 Responsive Design

🧾 Product detail view

🛒 Offcanvas cart UI
